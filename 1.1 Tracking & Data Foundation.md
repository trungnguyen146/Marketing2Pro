# Module 1: Tracking & Data Foundation

⏱ Thời gian: **1–2 tháng**

## 1.1 Digital Marketing Measurement

### 📚 Lý thuyết

Digital marketing khác marketing truyền thống ở chỗ:

**mọi thứ đều đo được**

Hệ thống đo lường gồm:

```
Traffic
↓
Event
↓
Conversion
↓
Revenue
```

Ví dụ:

```
1000 người vào web
↓
100 click nút
↓
20 người điền form
↓
5 khách mua
```

---

### 🧠 Cần hiểu

Phân biệt:

**Event**

```
user action
```

Ví dụ

```
click
scroll
video play
```

**Conversion**

```
action mang lại giá trị kinh doanh
```

Ví dụ

```
lead
purchase
signup
```

---

### 🛠 Bài tập

Chọn 1 website bất kỳ.

Liệt kê:

```
5 event
3 conversion
```

---

## 1.2 Meta Pixel

### 📚 Lý thuyết

Meta Pixel giúp:

```
track user behavior
optimize ads
retarget
```

Các event chuẩn:

```
PageView
ViewContent
AddToCart
InitiateCheckout
Purchase
Lead
```

---

### 🧠 Cần hiểu

Pixel giúp Meta hiểu:

```
ai là người có khả năng mua
```

Sau đó Meta tối ưu ads.

---

### 🛠 Bài tập

Vào **Meta Event Manager**

Xem:

```
event nào đang được track
```

Nếu có website riêng:

cài pixel và track:

```
Lead event
```

---

## 1.3 Google Tag Manager

### 📚 Lý thuyết

GTM giúp:

```
quản lý tracking code
```

Thay vì sửa code website.

---

### 3 thành phần chính

**Tag**

```
code cần chạy
```

**Trigger**

```
khi nào chạy
```

**Variable**

```
dữ liệu đi kèm
```

---

### Ví dụ

Click button:

```
trigger: click button
tag: GA4 event
```

---

### 🛠 Bài tập

Track:

```
click nút đăng ký
```

Gửi event sang GA4.
