# Module 6: Marketing Finance

⏱ **1 tháng**

---

## 6.1 CPL

Cost per lead

```
Ads spend / lead
```

---

### Ví dụ

Ads spend

```
$500
```

Lead

```
100
```

CPL

```
$5
```

---

## 6.2 CAC

Customer acquisition cost

```
marketing cost / customer
```

---

## 6.3 LTV

Lifetime value

```
tổng tiền khách chi
```

---

### 🛠 Bài tập

Giả sử:

```
ads spend = $2000
lead = 400
customer = 40
```

Tính:

```
CPL
CAC
```
