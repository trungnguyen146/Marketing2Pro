## 2.1 Traffic Source

### 📚 Lý thuyết

Traffic có nhiều nguồn:

```
Organic
Paid
Direct
Referral
Social
```

---

### 🧠 Ví dụ

```
Google search → organic
Facebook ads → paid social
```

---

```

```

```

```

---

## 2.2 Conversion Rate

### 📚 Lý thuyết

CVR =

```
Conversion / Traffic
```

Ví dụ

```
1000 traffic
50 lead
```

CVR

```
5%
```

---

### 🧠 Insight

Traffic nhiều chưa chắc tốt.

Quan trọng:

```
traffic chất lượng
```

---

### 🛠 Bài tập

Tính:

```
CVR landing page
```
